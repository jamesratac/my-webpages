About

Get Started with the DRG App

Welcome to the DRG App User Guide!

Within this manual, we will show you how to run the DRG app, which can calculate Adjusted Base Prices, and New Multiplying Factor that are in line with the Year 2 Full Implementation of Diagnosis Related Group (DRG) in Dubai, United Arab Emirates.


Download

Downloading the DRG App

First, you need to download the application program along with its components here.


Set Up

Extract File to Your Desktop Computer

Once downloaded, next step is to extract the file in your Desktop Computer.

Note: The extracted file should be located in your Desktop Computer for the application to run properly.

Your DRG app folder will look like this:

Prepare

Required Files

Now, before you run the DRG app, you need to prepare the following Microsoft Excel files:

    1. Provider Claim Items Data
    2. Provider's Agreed Tariff
    3. Provider Exclusion (if there is any)

Provider Claim Items data are being extracted from QlikSense BI.

For Provider's Agreed Tariff and Provider Exlusion worksheets, templates are provided, which can be accessed inside your DRG app folder under folders [020_agreedTariff] and [030_provExcl].
You need to have the data ready the same as the templates, else you might encounter an error when you run the app.

You may refer to Annex A to see the required fields for each of these files.


Run the App

Step-by-Step Guide for Running the App

INPUT Tab

Step 1. In the DRG app's INPUT tab, select a Provider Claim Items File in .xlsx format.

Step 2. Choose the Provider's Agreed Tariff.
Note: Make sure to save the Provider Price List in the [020_agreedTariff] folder prior to running the app.
Step 3. Select one or more Member Network/s from the list box selection.


EXCLUSION Tab

Step 4. In the DRG app's EXCLUSION tab, you can optionally include/exclude SAADA.
SAADA, by default, is excluded in the calculation. Tick the box if you want to include SAADA.
Step 5. There is an option to exclude Payers, Policies, Service Items, and/or Invoice Numbers.
Just enter the list of items you want to exclude in the ProvExcl.xlsx file. Link of the file provided in the textbox.

OUTPUT Tab

Step 6. In the DRG app's OUTPUT tab, enter the Output Filename.
App's output are saved into the [050_appOutput] folder by default.
Step 7. Click Finish.


App Output

Report Output Format

01_Summary – includes Adjusted Base Rate and Multiplying Factor, with and without High-Cost calculations

02_IPCases – details of in-patient claims with corresponding add on and inlier payments

03_DaycareCases – details of daycare claims with corresponding inlier payments

04_IPRejectedInvoices – lists all rejected IP claims

05_DaycareRejectedInvoices – lists all rejected IP claims
