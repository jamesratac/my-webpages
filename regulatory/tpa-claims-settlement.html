Overview

Overview

  • Deadline of Submission: Seven working days every after last day of the month
  • Scope: Previous Reporting Month, Claims under HAAD Licensing Authority - by PO Validation Date
Important Notes:
  • This is a regulatory requirement so make sure not to miss. Nextcare Compliance receives an email notification from Department of Health once this report has not been submitted after the deadline.
  • Make sure not to locked out the account by entering incorrect username/password. The credentials are linked to other Nextcare system, such as Greenrain.

Steps

Steps

  1. Download and save the data
    2. From: Qlik BI > Nextcare Claims Dashboard > Bookmarks = HAAD Processed & Ready for Settlement Claims To: \\ncu-tqa-019\DATA\james\__recurring-reports\haad-data-upload\HAAD UPLOAD YEAR 2022 Note: Create a new folder for the month, or if it's a new year, go one folder up (parent folder) and create a new HAAD UPLOAD YEAR 20XX.
  2. Filter the Provider PO Validation Month-Year. This is the only filter that you should be changing. Default filters are:
    • Licensing Authority = HAAD
    • ASOAP Type = Direct
    • Batch Reception Year <> Null
    • PO Delivert Date <> Null
  3. Go to Department of Health homepage: https://bpmweb.doh.gov.ae/UserManagement/MainPage.html
  4. Login using Nextcare credentials
    • Username = Nextcare
    • Password = to be provided by Helen Kay
  5. After logging in, go to: e-Insurance > TPA Forms
    6. If in case Finance team already uploaded their data, you will receive an error message = "Data already exists for this month!!!". If you encounter this error, go to (a); Else go to (b). a. Add the payers one-by-one by selecting the name of the payer in the dropdown menu and then clicking 'Add Insurer'. b. Go to e-Insurance > TPA Forms, and then select the latest report saved as draft.
  6. For each payer, click the gear icon to add the figures in the form.
  7. Once all data have been added, validate the figures by making sure that the totals from Qlik Sense and in the DOH Portal that you have just added are matching.
  8. Once confirmed that the totals matched, go to the bottom of the page and click Submit. Note: Never submit the report without data in Table 3 to 7, which are being completed by Finance department.
